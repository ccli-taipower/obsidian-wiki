# CKD 患者的壓力反射敏感度

壓力反射敏感度（baroreflex sensitivity, BRS）是評估自律神經調控血壓能力的關鍵指標。CKD/ESRD 患者的 BRS 普遍降低，與透析中低血壓（IDH）、心血管死亡率及血管鈣化密切相關。

## BRS 的生理學

- 壓力感受器位於主動脈弓和頸動脈竇的動脈壁內
- 血壓上升 → 壓力感受器發放增加 → 經迷走神經/舌咽神經傳入延髓 → 降低交感活性 + 增加副交感活性 → 心率下降 + 血管舒張
- BRS 量化為每 mmHg 血壓變化所引起的 RR interval 變化（ms/mmHg）

## CKD 患者 BRS 降低的原因

1. **血管鈣化與動脈硬化**：壓力感受器被鈣化的動脈壁包圍，機械感受功能受損
2. **尿毒素**：直接損傷自律神經
3. **RAAS 異常**：renin-angiotensin 系統過度活化
4. **貧血**：組織缺氧影響自律神經功能
5. **電解質異常**：鈣、鉀、鎂等失衡
6. **左心室肥厚**：心臟結構改變影響壓力反射弧的心臟端

## BRS 降低的臨床後果

- **透析中低血壓**：BRS 受損 → 超濾引起的血容量下降無法代償 → IDH
- IDH-prone + impaired BRS 的患者，CO 下降與 RBV 下降高度相關（r=0.94），代償機制完全失效
- **心血管死亡率增加**：BRS 降低是全因死亡的獨立預測因子
- **反覆心肌缺血**：透析中血流動力學不穩定 → myocardial stunning

## 測量方法

| 方法 | 優點 | 缺點 |
|------|------|------|
| Phenylephrine 注射 | 金標準 | 侵入性 |
| Finometer / Finapres | 非侵入、連續、多參數 | 設備成本 |
| Sequence method | 自發性、無藥物 | 需足夠 sequences |
| Delayed sequence method | 縮短記錄時間 | 較新、驗證中 |
| Heart rate variability | 簡便 | 間接指標 |

## 改善 BRS 的策略

- **運動訓練**：透析中 7 個月運動可改善 BRS +23%（Petraki 2008）
- **PTX**：減少血管鈣化 → 改善動脈硬化 → 可能改善 BRS
- **透析處方調整**：無醋酸鹽透析（paired haemodiafiltration）、夜間長時間透析

## 與 PTX 後低血壓的關聯

- PTX 後如果血壓急劇下降，BRS 受損的患者可能無法有效代償
- PTX 長期可能改善血管鈣化 → 改善 BRS → 改善透析耐受性
- 術前 BRS 評估可能有助於預測 PTX 後血流動力學反應
