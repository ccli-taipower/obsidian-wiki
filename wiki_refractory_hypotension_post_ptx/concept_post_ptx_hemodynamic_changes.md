# PTX 後血流動力學變化

副甲狀腺切除術（PTX）後的血壓變化呈現高度異質性，從漸進性輕度下降到急劇嚴重低血壓均有報告，機轉涉及細胞內鈣重置、血管壁鈣外流及心功能改變。

## 血壓變化的頻譜

| 模式 | 時間進程 | 幅度 | 代表文獻 |
|------|---------|------|---------|
| 漸進性下降 | 3-12 個月 | SBP -8 to -17 mmHg | Pizzarelli 1993, Sofronie 2018 meta-analysis |
| 完全正常化 | 數週至數月 | 從 200/130 降至正常 | Sofronie 2018 case |
| 急劇嚴重低血壓 | 數小時至數天 | 從 180/100 降至 60/40 | Leiba 2013 |
| 無變化 | — | — | 部分病例 |

## 預測因子

- **術前血壓**：術前越高，降壓幅度越大（Pizzarelli 1993: delta SAP = -0.54 × pre-PTX SAP + 49.3）
- **PTH 下降幅度**：PTH 下降越大，血壓變化越顯著
- **術前正常血壓者**可能出現低血壓，增加透析中低血壓風險

## 機轉假說

### 1. 細胞內鈣重置（Intracellular calcium resetting）
- PTH 急劇下降 → 停止促進鈣進入血管平滑肌 → 細胞內 Ca²⁺ 降低 → 血管舒張
- 血漿鈣同時下降（hungry bone syndrome）進一步減少細胞外鈣供應
- 解釋急性嚴重低血壓（Leiba 2013）

### 2. 血管壁鈣外流（Calcium efflux, Pizzarelli hypothesis）
- 長期副甲狀腺亢進 → 鈣沉積於血管壁
- PTX 後骨病變癒合 → 鈣從血管壁緩慢外流至骨骼
- 鹼性磷酸酶下降（反映骨癒合）與血壓下降相關
- 解釋漸進性血壓下降

### 3. 心功能改善
- PTX 後 LVEF 改善（Shih 2013: 53.6% → 59.3%）
- LVH 消退（Sofronie 2018: 172 → 116 g/m²）
- 動脈硬化改善 → 脈壓降低

### 4. 低血鈣誘發心衰竭（少數病例）
- 潛伏性心肌功能障礙被術後低血鈣「揭露」（Shinoda 1992）
- 鈣輸注可逆轉

## 臨床管理建議

- 術後密切監測血壓，特別是術前高血壓患者
- 預防性鈣輸注可減少急性血流動力學不穩定
- 降壓藥物需在術後積極調整（減量或停藥）
- 長期追蹤（至少 1 年）才能完整評估血壓效果
