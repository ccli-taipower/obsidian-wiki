# 分析：無移植情境下的預後推估與可修正因子

> ⚠️ 本分析為風險框架推估，非精確預測。個體差異極大，最終判斷需心臟科 + 腎臟科整合評估。

完整病人摘要見 [analysis_refractory_idh_treatment_plan](analysis_refractory_idh_treatment_plan.md)。

## 基線：透析患者存活率

| 數據來源 | 年齡層 | 5 年存活率 | 10 年存活率 |
|---------|------|---------|---------|
| USRDS（美國） | 45-54 歲 HD | ~50-55% | ~25-30% |
| 台灣腎臟醫學會 | 類似年齡層 | 略優於美國 | — |

基線預期：一般 48 歲透析病人，中位存活約 5-7 年。但個人化因子會大幅偏離。

## 個人化風險分層

### 不利因子（縮短壽命）

| 因子 | 嚴重度 | 機轉 |
|------|--------|------|
| **SSS=16 嚴重可逆缺血** | 🔴 極高 | 年心臟事件率估 10-15%；未血管重建時 5 年心臟存活率差 |
| **RCA CTO + 多血管 CAD** | 🔴 高 | 缺血範圍跨 LAD/LCX/RCA territory（核醫 2025/08） |
| **頑固性 IDH** | 🟠 中高 | 反覆 myocardial stunning → 加速心肌損傷；腦部已有 mild WMH |
| **血管鈣化**（主動脈 + 冠脈） | 🟠 中高 | 進行性；iPTH 過度抑制 + 鈣 binder 1,200 mg/日持續加重 |
| **Adynamic bone**（iPTH 17.5 + ALP 61） | 🟠 中 | 鈣無法進入骨骼 → 血管沉積 |
| HD 11 年 | 🟡 中 | 累積尿毒素、血管通路消耗 |
| AVG 反覆狹窄 | 🟡 中 | 通路壽命有限；2026/02 PTA 90% |
| **FlowPRA Class I 44%** | 🟡 中 | 中度致敏，移植配對池縮小；MFI 563 偏低是緩衝 |

### 有利因子（延長壽命）

| 因子 | 意義 |
|------|------|
| **年齡 48（相對年輕）** | 比 65+ 透析患者有更大生理儲備 |
| **ADPKD 病因** | ADPKD 透析患者存活率優於糖尿病腎病變（較好營養、較少全身性微血管病變） |
| **ADPKD 保留 EPO 分泌** | 透析 11 年不用 EPO、不輸血、Hb 自維持 12.3 → 囊腫壁細胞仍能分泌 EPO（ADPKD 特殊優勢）；不再輸血 = PRA 不會繼續升高 |
| **Albumin 4.2** | 營養狀態好是透析患者存活的強力保護因子 |
| **EF 73.7%** | 收縮功能保留（雖有 HFpEF 但收縮儲備仍在） |
| **CTR 0.4** | 心臟結構尚未不可逆損傷 |
| **仍正常上班** | 功能狀態好，強力預後指標 |
| **顱內動脈瘤陰性** | ADPKD 最怕的致命併發症已排除 |

## 最大單一威脅：SSS=16

核醫 SSS=16 + ΔTPD=16% 是整個預後的關鍵決定者。

在透析患者中：
- SSS ≥ 13 屬於 severe abnormal
- 嚴重可逆缺血且未血管重建 → 年心臟死亡率估 8-15%
- 5 年內發生重大心臟事件（MI、心因性猝死）風險非常高
- 缺血範圍跨多個 territory 進一步加重風險

**如果不處理 CAD，心臟是最可能的死因，時間框架可能比一般透析存活預期更短。**

## 情境分析

| 情境 | 預估中位存活 | 主要風險 |
|------|-----------|--------|
| **什麼都不改變** | **3-5 年** | 心因性猝死或急性冠脈事件；IDH 持續 stunning → 心功能惡化 |
| **處理可調因子但不處理 CAD** | **5-7 年** | 停沛暢 + iPTH 回升 + ferric citrate 改善 IDH，但冠脈缺血仍是定時炸彈 |
| **積極處理 CAD + 可調因子** | **7-10+ 年** | RCA CTO 開通 + LCX 再評估 → 大幅降低心臟事件風險；移植可能性也大增 |

## 預後改變槓桿排序

| 排序 | 介入 | 預後影響 | 詳見 |
|------|------|--------|------|
| **1** | **腎臟移植** | 5 年存活率從 ~50% → ~85% | （本分析排除此選項） |
| **2** | **RCA CTO 開通 + 冠脈完全血管重建** | 直接降低最大死因風險 | [analysis_cad_contribution](analysis_cad_contribution_to_refractory_idh.md) |
| **3** | iPTH 回升至 150-300 | 改善 IDH + 減緩血管鈣化 | [analysis_refractory_idh](analysis_refractory_idh_treatment_plan.md) |
| **4** | 停沛暢 + 換 ferric citrate | 改善 IDH + 減少鈣負擔 + 補鐵 | [analysis_iatrogenic_factors](analysis_iatrogenic_factors_in_refractory_idh.md) |
| **5** | 重啟 BB（bisoprolol） | HFpEF + ischemia 雙重保護 | [analysis_cad_contribution](analysis_cad_contribution_to_refractory_idh.md) |

## 移植免疫障礙：FlowPRA 數據（2026/03/02）

雖然本分析以「無移植」為前提，但移植仍是排序第一的預後改變槓桿。以下數據直接影響移植可行性。

### FlowPRA 結果

| 項目 | 數值 | 解讀 |
|------|------|------|
| **FlowPRA Class I (%)** | **44%** | 中度致敏（20-80% 區間） |
| FlowPRA Class I MFI | 563 | 抗體強度偏低（<1,000） |
| **FlowPRA Class II (%)** | **11%** | 輕度致敏（10-20% 區間） |
| FlowPRA Class II MFI | 355 | 抗體強度低 |

同日其他數據：Potassium K 6.1（115/02/12，與既有 K 6.0 趨勢一致）

### 臨床意義

**Class I PRA 44% = 每 100 個潛在捐贈者中，約 44 個因 HLA Class I 抗體不合而配對失敗**。等待時間顯著延長。

致敏原因：透析 11 年間歷次輸血累積（手術、急性事件）。ADPKD 本身不增加致敏風險。

**重要更新**：病人目前不用 EPO 也不輸血（ADPKD 囊腫壁細胞仍能分泌 EPO，Hb 12.3 自維持）。HLA 抗體由記憶 B 細胞維持，不會自行消失，但**不再輸血 = PRA 不會繼續升高**。

### 相對有利點

MFI 563 / 355 均偏低 → 抗體存在但**強度不高**：
- 虛擬交叉試驗（virtual crossmatch）結果可能比 PRA 百分比樂觀
- 若做 single antigen bead（SAB）細分，部分抗體 MFI 可能低到可接受範圍
- 不是所有 44% 的不合都會造成超急性排斥

### 移植的兩道關卡

| 關卡 | 狀態 | 對策 |
|------|------|------|
| **① 心臟（SSS=16）** | 嚴重可逆缺血，移植前需先處理 | RCA CTO 開通 + LCX 再評估 |
| **② 免疫（PRA 44%）** | 中度致敏，配對池縮小 | SAB 細分 + acceptable mismatch + 活體評估 |
| **③ 解剖（ADPKD 巨腎）** | 雙側腎臟明顯增大（腹部 MRI 2025/12），佔據腹腔空間 | 評估原生腎切除時機 + 腎臟體積測量 |

三個關卡都需要**主動積極處理**，不能被動等待。

### 免疫對策建議

1. **確認是否已做 SAB（單抗原珠檢測）** → 精確辨識是哪幾個 HLA 抗原有抗體，排除「可接受的不匹配」
2. **加入 acceptable mismatch program**（台大/長庚等移植中心）→ 擴大有效配對池
3. **評估 desensitization（脫敏治療）可行性** → 若出現合適捐贈者但交叉試驗弱陽性，可用 IVIG / rituximab / plasmapheresis
4. **活體捐贈評估** → 如有家人願意，可針對特定 donor 交叉試驗；44% 對單一已知捐贈者不一定是障礙
5. **減少未來致敏** → 盡量避免不必要輸血（EPO + IV iron 維持 Hb；ferric citrate switch 有助於此）

### 解剖障礙：ADPKD 巨腎與原生腎切除

腹部 MRI（2025/12）顯示雙側腎臟明顯增大。ADPKD 腎臟持續生長，造成移植的解剖障礙：

**為什麼是障礙：**
- 移植腎放在髂窩（骨盆腔），巨大原生腎佔據腹腔空間 → 移植腎放不進去或被壓迫
- 腫大腎臟壓迫髂動靜脈 → 血管接合困難、外科視野差
- 術後巨大腎臟持續生長 → 壓迫移植腎 → 影響移植腎功能

**原生腎切除（Native Nephrectomy）時機選擇：**

| 時機 | 優點 | 缺點 |
|------|------|------|
| **移植前切除** | 騰出空間，移植手術簡化 | 無腎期（完全依賴透析）、等待期感染風險增加 |
| **移植同時切除** | 一次手術解決 | 手術時間長、出血風險高 |
| **移植後切除** | 先確保移植成功 | 若空間不夠則無法執行移植 |

台灣多數移植中心傾向移植前或同時切除，取決於腎臟體積（通常以 TKV 總腎臟體積評估）。

**對策：**
1. 確認腹部 MRI 2025/12 是否有報告 TKV（total kidney volume）
2. 與移植外科 / 泌尿外科討論原生腎切除的時機與方式
3. 若決定移植前切除，需考量：切除後完全無殘餘腎功能、術後恢復期、與心臟手術的排程協調

## 核心訊息

不是「還能活多久」的問題，而是「有哪些可以做的事能改變軌跡」。

這位病人的**可調空間不小**：
- 最大威脅（SSS=16）是**可逆缺血**（心肌仍存活）→ 血管重建有機會大幅改善
- 多個低成本高效益介入尚未執行（停沛暢、iPTH 回升、ferric citrate、BB）
- 功能狀態好（正常上班）+ 營養好（Alb 4.2）→ 承受積極治療的本錢仍在
- 心臟結構尚未不可逆（CTR 0.4、EF 73.7%）→ 時間窗口還沒關閉

**但時間窗口有限**——SSS=16 每多一年未處理，心肌從可逆缺血進展到不可逆壞死的風險就在累積。

## 病人自我管理建議（仍正常上班的前提下）

病人目前功能狀態良好（正常上班），這本身是強力預後保護因子。但 SSS=16 + 頑固性 IDH 意味著日常生活需要主動管理風險。

### 1. 工作安全

- 確認工作性質是否涉及駕車、高處作業、操作機具 → IDH 造成的「眼前發黑」有暈厥風險
- **透析隔天（二、四、六早上）是最脆弱時段** → 避免安排需要高度專注或體力的工作
- 隨身攜帶**緊急聯絡卡**（透析患者身份、用藥、AVG 側肢體不量血壓不抽血）

### 2. 心臟猝死預警

SSS=16 嚴重可逆缺血 = 隨時可能發生急性冠脈事件。必須辨識：
- 胸悶、冷汗、下巴/左臂痛、突然喘不過氣 → **立刻叫 119，不要忍**
- 透析中血壓掉到比平常更低且合併胸悶 → 主動告知護理師（不是「老問題」）
- 與心臟科討論是否備用舌下 NTG（但 IDH 體質需極度謹慎使用）

### 3. 飲食管理 — K 6.0 是最緊急的自我管理項目

K 6.0 + active ischemia = 心律不整風險明顯升高：

| 建議 | 說明 |
|------|------|
| **楊桃絕對禁止** | ADPKD + 透析 = 神經毒性 |
| 高鉀水果減量 | 香蕉、奇異果、番茄、柳橙 |
| 蔬菜先燙再炒 | 溶出 30-50% 鉀 |
| 湯品少喝 | 鉀溶在湯裡 |
| **不要用低鈉鹽** | 低鈉鹽 = 高鉀鹽 |
| IDWG 控制 | 目前 5%，目標 <4%，控制兩次透析間飲水量 |

### 4. 運動建議

[Petraki 2008](src_petraki2008_exercise_training_brs.md) 顯示透析中運動改善壓力反射敏感度（BRS +23%），直接對抗 IDH 的自主神經失調面向。

| 類型 | 建議 |
|------|------|
| **透析中腳踏車** | intradialytic cycling，跟洗腎室確認設備 |
| 非透析日有氧 | 中等強度快走 30 分鐘，**避免透析隔天早上** |
| 避免 | 重量訓練到 Valsalva（血壓劇烈波動 + ischemia 風險） |
| 注意 | 運動前不要空腹（低血糖 + IDH 體質 = 暈厥風險） |

### 5. 最重要的單一行動：推動心臟科再評估

- 「還能上班」≠ 心臟沒事 — SSS=16 是影像硬證據，心肌正在缺血
- 目前是**可逆缺血**（心肌還活著）→ 血管重建有意義
- 每多拖一年，可逆→不可逆壞死的風險在累積
- **這也是移植的門票** — 腎移植評估時 SSS=16 會被要求先處理
