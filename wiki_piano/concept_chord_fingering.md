# 和弦指法 (Chord Fingering)

三和弦在不同轉位下的標準指法，適用於所有大小三和弦。

## 標準三和弦指法

| 轉位 | 左手 | 右手 |
|------|------|------|
| 原位 (Root) | 5-3-1 | 1-3-5 |
| 第一轉位 | 5-3-1 | 1-2-5 |
| 第二轉位 | 5-2-1 | 1-3-5 |

## 八度和弦（四音）

| 手 | 指法 |
|------|------|
| 左手 | 5-4-2-1 |
| 右手 | 1-2-3-5 |

## 左手例外規則

- 需要寬跨距時：5-3-1 → **5-2-1**（如 A 小調第二轉位）
- 首音為黑鍵時：用 **4** 開頭（如 D 大調第一轉位 F#-A-D = **4-2-1**）← **V6 已實作**：`LH_PINKY_BLACK_KEY_PENALTY = 2.0`
- 四音和弦 3 指有其他用途時：5-3-2-1 → **5-4-2-1**

## 音域適應

同一和弦在不同音域需要不同指法（Penelope Roskell）：
- **中音域**：1-3-5（標準）
- **低音域**：避免拇指 → **2-3-5** ← V6：`REGISTER_LOW_MIDI=48`，RH f1 < C3 → 懲罰（2音和弦有效；3音和弦因強制含 f1 效果有限）
- **高音域**：避免小指 → **1-2-4** ← **V6 已實作**：`REGISTER_HIGH_MIDI=84`，RH f5 > C6 → `REGISTER_OUTER_PENALTY=1.5`

原則：保持手指與前臂對齊，避免手腕扭轉。
注：LH 低音域 f5 = 正常 bass 行為，不懲罰。LH 高音域（f1 > C6）同步懲罰。

## 一般原則

- 無論黑白鍵，標準指法均適用
- 八度和弦常用於增強音量感
- 右手第一轉位使用 1-**2**-5（而非 1-3-5），因中間音與底音距離較近
- 3 vs 4 的選擇：以手指間距均勻為原則（Graham Fitch）

## 分解和弦（Broken Chord / Arpeggio）

分解和弦是將和弦音依序彈出，但**指法原則與塊狀和弦相同**，不使用音階 relay 公式：

- 以 C(5,k) 組合分配手指：右手拇指壓最低音，左手拇指壓最高音
- 相鄰音程 ≥ 3 半音 = 和聲跳躍（非音階步進），應視為和弦處理
- 標準三音分解和弦：RH 上行 1-3-5，LH 上行 5-3-1

**識別條件**（同一小節內）：
1. 連續 ≥ 3 個單音
2. 所有相鄰音程 ≥ 3 半音
3. 音高方向一致（純上行或純下行）

**例**：BWV 775 m3 RH — F4-A4-D5（D minor 第一轉位分解）→ 1-3-5
