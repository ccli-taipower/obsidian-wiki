# 鋼琴家手部跨度測量與人體工學鍵盤

3 篇論文提供了鋼琴家手部跨度的**量化數據**和人體工學鍵盤的**臨床實證**。

## 1. Boyle (2013) — 縮尺鍵盤的生物力學效益

### 調查結果

22 位使用縮尺鍵盤的鋼琴家（1-5 跨度 ≤ 8 英寸），報告感知改善：

**100% 報告「相當」或「顯著」改善**的技能：
- 按住音符的能力（不再提前釋放用踏板掩蓋）
- 力量感

**>80% 報告改善**的技能：
- 快速八度/大和弦段落（95%）
- 技術困難段落的學習時間（90%）
- 碎八度（86%）、碎和弦/琶音（86%）
- 安全感（86%）、非標準指法（85%）

### Ortmann 生物力學原理（1929）

⭐ 系統性回顧 Otto Ortmann 的《鋼琴技巧的生理力學》對小手的影響：

**1. 手指外展與彎曲**
- 最大能量傳遞需要手臂-手-指一直線（直線骨骼位置）
- 手指外展越大 → 偏離前臂軸線越多 → 動能損失越多 → 音量和控制降低
- 彎曲手指觸鍵力 = 平直手指的 **5/3 倍**（力臂反比關係）
- 小手被迫使用平直手指觸及八度 → 力量和控制雙重損失

**2. 手腕位置**
- 小手彈八度時傾向抬高手腕（增加力量、避免碰鄰鍵）
- 但抬高手腕 → 拇指和 5 指失去獨立性，變成「戳鍵器」
- Ortmann：跨度 ≤ 7.5 英寸的鋼琴家在白鍵邊緣彈八度，「不實際且不舒適」

**3. 側向移動**
- 小手在碎八度、琶音、跳進中需要更多側向移動
- 更多移動 = 速度損失 + 音色控制損失 + 準確度降低

**4. 前後移動**
- 半音八度需要手在黑白鍵之間大幅前後移動
- 跨度 < 8.5 英寸的鋼琴家無法舒適地跨越黑鍵

**5. 敲擊 vs 非敲擊觸鍵**
- 小手更常使用敲擊觸鍵（從高處落下觸鍵）以達到速度
- 但敲擊觸鍵 = 音色控制降低 + 不必要的機械噪音

### 三級效益模型

| 層級 | 內容 |
|------|------|
| Level 1：具體任務 | 八度、大和弦、碎八度、音階、指法選擇 |
| Level 2：演出成果 | 音色、速度、力度控制、準確度、連奏 |
| Level 3：鋼琴家體驗 | 安全感、舒適度、曲目選擇、縮短學習時間 |

## 2. Boyle, Boyle & Booker (2015) — 鋼琴家手部跨度的性別與族裔差異

### 研究規模

⭐ **最大規模的鋼琴家手部跨度研究**：
- **473 位成年鋼琴家**（314 女 / 159 男）
- 49 位兒童/青少年（6-17 歲）
- 216 位非鋼琴家大學生（對照組）
- 2011-2014 年收集，主要為澳大利亞鋼琴家

### 核心數據

**全體成年鋼琴家（右手 1-5 跨度）**：

| 統計量 | 數值 |
|--------|------|
| 範圍 | **6.4 - 10.8 英寸**（16.3 - 27.4 cm） |
| 平均 | **8.2 英寸**（21 cm） |
| 中位數 | 8.2 英寸 |
| 標準差 | 約 1 英寸 |

**性別差異**：

| 指標 | 男性 | 女性 | 差距 |
|------|------|------|------|
| 1-5 跨度平均 | 約 8.9" | 約 7.9" | **~1 英寸（2.5 cm）** |
| 最小值 | 7.8" | 6.4" | — |
| Q1（75% 大於此值） | 8.5" | 7.5" | — |

- 男性最小跨度（7.8"）≈ 女性平均（7.9"）
- **83.7% 的男性**跨度 > 女性 Q3（8.3"）
- 性別差異 p < 0.0000

**族裔差異**：

| 比較 | 差距 | 顯著性 |
|------|------|--------|
| 白人女性 vs 亞洲女性 | ~0.4"（1 cm） | p < 0.05 |
| 白人男性 vs 亞洲男性 | 較小差距 | 部分顯著 |

### 分布特徵

1-5 跨度呈**雙峰分布**（peaks at ~8" and ~9"），反映性別差異。

### 「小手」的定義建議

- 需要 **7.5 英寸**（19 cm）主動跨度才能在白鍵上舒適彈八度
- 需要 **8.5 英寸**（21.5 cm）才能以 3-4 指彈 legato 八度
- 約 **87% 的女性鋼琴家**和 **24% 的男性鋼琴家** 1-5 跨度 < 8.5"

### 知名鋼琴家的手

高成就鋼琴家的手傾向**大於平均**——鍵盤設計偏向大手的歷史偏見得到統計證實。

## 3. Yoshimura & Chesky (2009) — DS 15/16 人體工學鍵盤的臨床應用

### 研究設計

- 35 位北德州大學鋼琴主修學生（27 女 / 8 男，57% 亞洲裔）
- 兩架 Steinway Model L：一架標準（188mm 八度），一架 DS 15/16（174mm 八度）
- 練習指定曲目 2 週後，分別在兩架鋼琴上演奏
- 使用 VAS（視覺類比量表）測量疼痛和緊張度

### 人體測量數據

| 測量 | 最小 | 最大 | 平均 | 標準差 |
|------|------|------|------|------|
| 右手跨度 | 183mm | 250mm | 209.7mm | 17.2mm |
| 左手跨度 | 181mm | 250mm | 212.4mm | 17.7mm |
| 手長 | 148mm | 202mm | 174.4mm | 13.0mm |

### 核心結果

| 指標 | 188mm 鍵盤 | 174mm 鍵盤 | 顯著性 |
|------|----------|----------|--------|
| 演奏中疼痛 | 2.07 | 0.96 | p < 0.01 |
| 演奏中緊張 | 3.17 | 2.20 | p < 0.05 |
| 演奏前後疼痛變化 | +0.71（增加） | -0.13（減少） | p < 0.05 |

### 手跨度與疼痛的相關性

- 188mm 鍵盤：手跨度與疼痛**顯著負相關**（r = -0.63, p < 0.01）
- 174mm 鍵盤：相關性仍存在但**斜率較低**（r = -0.52, p < 0.01）
- 結論：縮尺鍵盤**不能消除**手部大小對疼痛的影響，但能**顯著降低**

### 歷史鍵盤尺寸

從 75 架歷史鋼琴的測量：
- 1720 年：178mm 八度
- 1784-1800：182mm
- 1826-1850：186mm
- **1876 年後**：188mm（標準化）

## 與 V6 DP 的關聯

| 發現 | V6 DP 啟示 |
|------|----------|
| 1-5 跨度 6.4"-10.8"（4.4" 差距 ≈ 4.5 白鍵） | FINGER_COMFORT_SPAN 不應是固定值 |
| 87% 女性 < 8.5"、24% 男性 < 8.5" | 預設跨度表假設「平均男性」→ 多數女性需要縮放 |
| Ortmann：外展降低力量和控制 | V6 的 span_cost 正確地懲罰伸展 |
| 174mm 鍵盤降低疼痛和緊張 | 不同鍵盤尺寸可能需要不同的跨度表參數 |
| 雙峰分布（男 ~9" / 女 ~8"） | 可考慮 2 套預設跨度表（而非單一 ×0.9 縮放） |
