# 極簡主義指法 (Minimalist Piano Fingering)

由 Cory Hall (BachScholar) 於 2024 年提出的指法哲學，挑戰兩百年來的傳統 C 大調指法模型。

## 核心主張

> 最佳指法是用**最少手指**且**最多拇指穿越**的指法。

- 傳統 C 大調指法（RH: 123-12345）不是最優解
- F 大調指法（RH: 1234-1234）應取代 C 大調成為「預設」指法
- 小指（5）是最弱的手指，應盡量省略
- 「學術正確」的指法往往是實踐中最難的指法

## 與傳統的對比

| 傳統 | 極簡主義 |
|------|----------|
| 用 5 指，123-12345 | 用 3-4 指，省略 5 |
| 減少拇指穿越 | 增加拇指穿越 |
| 訓練弱指（4, 5）平衡 | 最大化強指（1, 2, 3）優勢 |
| C 大調指法出現 10/24 次 | C 大調指法僅出現 1/24 次 |

## 實例

### Bach Invention No.4 主題
- 8 種可能指法中，傳統（用 5 指）是最難的
- 極簡版（只用 1-2-3）最清晰、最均勻

### Clementi Sonatina No.1 開頭
- 傳統 5 指版需處理 3-4-3-4-5-4-3 的弱指序列
- 3 指版只用 1-2-3 即可完成，且音色更均勻

### Clementi Sonatina No.3 C 大調音階
- 傳統：123-12345，拇指在 F 上造成不必要的重音
- F 大調指法：1234-1234，分為均等的 4+4 組
- 極簡：只用 1-2-3，最清晰

## 適用場景

- 短而快的音階片段
- 方向改變的經過句
- 三連音節奏型
- 不一定適用於長音階跑動或慢速抒情段落
