# Pedal Technique（踏板技法）

鋼琴踏板（sustain pedal）不是 0/1 開關，而是連續光譜的音色控制工具。高階演奏中踏板技法與指法同等重要——踏板決定了聲音的空間感、色彩融合度、以及音符之間的連接方式。

## 基本原理

### 延音踏板（Sustain Pedal）的物理機制

踩下延音踏板 → 制音器（damper）全部離開琴弦 → 所有已發聲或受共鳴激發的弦持續振動。放開踏板 → 制音器落回琴弦止音。

### 粗弦/細弦差異——半踏板的物理基礎

鋼琴低音弦粗、高音弦細，這個物理差異是進階踏板技法的關鍵：

- **低音粗弦**：振動時慣性大，制音器輕觸仍會繼續振動
- **高音細弦**：振動慣性小，制音器輕觸幾下即可止住

**實驗**：同時彈低音和高音，踏板輕輕放到一半 → 高音幾乎消失，低音仍然清楚。

這意味著可以**選擇性地保留低音同時清除高音和聲**——不需要第三隻手或腳。

## 進階踏板技法

### 1. 半踏板（Half Pedal）

踏板不完全踩到底或不完全放開，制音器輕觸琴弦：

- 高音被止住，低音持續
- 用於需要保持低音持續（bass pedal point）但上方和聲需要清晰分離的段落
- 深度可以精細控制——不是「半」而是「任意比例」

### 2. 延遲換踏板（Delayed Pedal Change）

換和聲時**故意不立即放開踏板**，讓前後兩個和聲短暫重疊，經過一段「髒」的時間後再慢慢變乾淨：

**三種層次的對比**：
- 不換踏板 = 所有聲音混在一起 → 太髒太混濁，不舒服
- 立即換踏板 = 和聲乾淨切換 → 乾淨但失去靈魂，失去鋼琴音樂的魔法
- 延遲換踏板 = 兩個和聲如化學反應般漸變 → **介於髒與淨之間的曖昧音色，是鋼琴聲音最美的時刻**

厲害的鋼琴家知道如何拿捏乾淨與不乾淨之間的聲音，創造聲音的魔法。

**踏板的速度也有差別**：可以慢慢踩下去、慢慢放開；在中間空間游移；輕碰一下再上來——不是只有快速踩放。

### 3. 踏板延續隱藏音（Sustained Resonance of Unstruck Notes）

蕭邦 Op.9-1 降 D 大調段落的經典案例：

- 左手連續十幾小節彈同一和弦，右手加入降 Do
- 降 Do 有強烈欲望要解決到降 C，但蕭邦始終不讓它解決
- 降 Do 不斷反覆後在中途突然消失
- 關鍵：蕭邦在譜上標記**不放踏板**
- 降 Do 雖不再被彈，但透過踏板共鳴仍存在——非常小聲但未完全消失
- 高音旋律面無表情地演奏遠方號角聲，降 Do 的共鳴默默延續在背景中
- 效果：遠方的聲音、超脫到另一個世界的意境

蕭邦用踏板延續了一個不再被彈出的音符，讓它雖然不被彈但仍存在於共鳴之中。蕭邦對踏板運用出了名的細緻。

### 4. 漸進放開（Gradual Release）——聲音錯覺

低音逐漸消失（而非瞬間切斷）→ 即使低音最終完全消散，聽者**不會察覺斷裂**。

對比實驗：
- 全放踏板 → 低音「噗」瞬間消失 → 觀眾立刻發現
- 慢慢放 → 低音逐漸消失 → 觀眾誤以為它還在

這是利用人耳對漸變消失不敏感的特性，製造聲音「錯覺」。

## 德布西《月光》的踏板示範（四個場景）

### 場景 1：開頭和聲色彩漸變

前 4 小節每次換和聲用延遲換踏板：
- 第一個和聲（乾淨大三和弦）到第二個和聲（減七和弦）
- 不馬上放踏板，讓兩個和聲短暫重疊
- 然後慢慢變乾淨——如化學反應
- 一路都可以這樣踩

### 場景 2：烏雲與星星——半踏板的極致運用

低音極低（接近鋼琴最底端）如烏雲籠罩一切，上方高音如星星閃爍：
- 需要：保留低音持續 + 上方和聲清晰分離
- 問題：如果一直踩踏板 → 所有音混在一起很髒；如果一直換踏板 → 低音不見
- 解答：利用粗弦/細弦差異——踏板輕放到一半，高音被清除，低音仍持續
- 譜上德布西要求低音一直延續，不能用換踏板讓它消失

### 場景 3：下行到上行——踏板漸進消失

和聲一路從高往下（如攝影機從月亮下移到街道），到最底部出現低音後，方向轉上。
- 到達頂峰時踏板逐漸消失
- 效果：爬到山頂抬頭仰望 → 天上都是星星 → 「哇」的開闊感
- 踏板的逐漸消失製造空間感的變化

### 場景 4：中段——虛無飄渺的紗簾

六音一組的分解和弦如飄在空中的紗布窗簾被風吹動：
- 不可彈得太實太大聲
- 和聲每次都在細緻變化（三度關係上行 → 瞬間飄到另一個世界）
- 踏板要配合這些和聲變化做出細緻的情緒細節

## 與鋼琴指法的關聯

踏板與指法的關係是鋼琴家之間意見分歧最大的主題之一（見 [src_talking_fingers_interview](src_talking_fingers_interview.md)）。

### 踏板減少 legato 指法需求

有踏板延音時，手指不必始終保持 legato 連接（踏板會橋接音符之間的空隙）。這意味著：
- 可選擇生物力學更優但不完全 legato 的指法
- 某些需要 finger substitution 的段落，有踏板時可以簡化

### 無踏板段落需要 finger substitution

沒有踏板延音時，保持長音需要 silent key change（在按鍵不放的情況下換手指）。見 [concept_finger_substitution](concept_finger_substitution.md)。

### 踏板掩蓋 vs 踏板輔助

**所有鋼琴家的共識**（來自 Talking Fingers 訪談）：用踏板來**掩蓋**糟糕指法是不可接受的。踏板應該是音樂表現的工具，不是彌補技術缺陷的捷徑。

### V6 DP 不建模踏板

目前 V6 DP 假設純手指 legato，未考慮踏板對指法選擇的放寬效果。這是一個已知的限制：
- 有踏板的段落，DP 可能過度偏好 legato 指法（因為不知道踏板會橋接）
- 無踏板段落，DP 的行為更接近實際需求

**Talking Fingers 訪談中的分歧**：
- 極端 A：踏板不影響指法，兩者完全獨立
- 極端 B：踏板改變需要的連奏程度，因此影響指法選擇
