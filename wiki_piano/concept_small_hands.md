# Small Hands（小手鋼琴家）

小手是鋼琴演奏中被長期忽視但影響深遠的議題——標準化鍵盤歧視手較小的演奏者，導致傷害、曲目限制和情緒壓力。

## 定義

小手是**自定義**的狀態（Deahl & Wristen, *Adaptive Strategies for Small-Handed Pianists*, 2017）。影響因素包括：

- 手指長度
- 手指間距（spread）
- 手掌寬度
- 手指寬度
- 拇指到小指跨度（1-5 finger span）
- 舒適彈奏八度或九度的能力
- 關節和手部的靈活性
- 上肢肌肉組織

無法在中性或近中性姿勢下演奏全音階五音音階，可作為小手的功能性定義（Chong 2021）。

## 標準鍵盤的歷史

- **1860 年代之前**：鍵盤尺寸多樣（如弦樂器可選擇不同尺寸的琴弓、管樂器可選擇不同吹嘴）
- **1860 年代之後**：標準化為 **6.5 英寸八度**（165mm）——消除了選擇的自由
- 標準化深化了性別、種族和社經地位的不平等

## 手部大小的性別與族裔差異

2015 年研究（Boyle et al.，473 位成年鋼琴家）：

| 比較 | 差距 |
|------|------|
| 男性 vs 女性 1-5 跨度 | 男性平均大 **1 英寸（2.5cm）** |
| 女性白種人 vs 女性亞洲人 | 白種人平均大 **0.4 英寸（1cm）** |
| 國際知名鋼琴家 vs 平均男性 | 知名鋼琴家手大於平均 |

## ESPK（人體工學縮放鋼琴鍵盤）

- 由 David Steinbuhler（賓州）生產數十年
- 提供多種尺寸，可**客製化安裝**在任何現有樂器上（包括 Steinway Model D）
- 但在鋼琴家和教育者之間**幾乎無人知道**
- 鍵盤製造業支持現狀：成功演奏家（大手）的背書 + 標準化生產的成本優勢

## 六大障礙

| 障礙 | 說明 |
|------|------|
| **1. 觸及（Reach）** | 寬和弦/琶音的跨距困難是最明顯的障礙。即使較小的音程，伸展也消耗更多體力。例：Rachmaninoff Op.18、Schubert D.899 |
| **2. 音質控制** | 手部伸展越大 → 控制力越差 → 聲部導引、大聲、legato 連接均受影響 |
| **3. 疲勞** | 持續伸展、跳躍、不自然角度；中性姿勢的微小偏離即可加速疲勞。例：Liszt Sonata in B mm.582-594（ff 八度 + stringendo） |
| **4. 曲目限制** | 浪漫時期以後大量需要大手的曲目；女性可能「偏好」Bach、Mozart、Debussy 以保護身體 |
| **5. 適應性** | 依賴他人鋼琴（觸鍵重量、深度各異）；重鍵盤可能摧毀小手鋼琴家精心安排的效率策略 |
| **6. 心理壓力** | 音樂妥協帶來的情緒壓力。Deahl & Wristen：「許多小手演奏者在沉默中受苦」 |

## 傷害統計

- 40-60% 鋼琴家有演奏相關肌肉骨骼疾病（PRMDs）
- Chi et al. 2020 系統回顧：**10/13 篇鋼琴研究**確認手較小 → 傷害風險較高
- 僅 2 篇發現手部大小不顯著影響 PRMDs
- 常見傷害：過度使用、腱鞘炎、重複性勞損、肌肉疲勞

## 應對策略

| 策略 | 說明 |
|------|------|
| **替代指法** | 使用非標準指法避免過度伸展（如 Roskell 的音域適應指法：低音域用 2-3-5 取代 1-3-5） |
| **手部重分配** | 將一隻手的音符轉移給另一隻手（見 [[concept_hand_distribution]]） |
| **滾奏和弦** | 將同時和弦改為快速琶音，減少瞬間跨度需求 |
| **省略音符** | 在不影響和聲本質的前提下省略和弦中的某些音 |
| **踏板使用** | 用踏板延續音符，減少手指必須同時按住的音 |
| **曲目選擇** | 選擇適合手型的曲目（但這限制了職業發展） |
| **聽覺想像（Audiation）** | Seymour Fink：音樂意圖是防止傷害的保護機制——先聽到聲音再演奏 |

## 與 V6 DP 的關聯

V6 DP 的跨度表（`FINGER_COMFORT_SPAN`）基於 Parncutt 1997，假設「平均成年男性鋼琴家」：

- Parncutt 自己建議：**乘以 0.9** 即可適應較小的手
- V6 目前的跨度值是**固定的**，不因手部大小調整
- 這意味著 V6 對小手鋼琴家可能產生**過度伸展**的指法建議

### 可能的改進方向

1. **手部大小參數**：新增 `HAND_SIZE_SCALE`（預設 1.0），乘以所有跨度值
   - 小手：0.85-0.90
   - 平均：1.0
   - 大手：1.05-1.10
2. **自動偵測**：若 MusicXML 中有大量超過 MaxComf 的和弦，自動建議縮放
3. **跨度表也影響**：`FINGER_MAX_SPAN` 和 `FINGER_MIN_SPAN` 同樣需要縮放
