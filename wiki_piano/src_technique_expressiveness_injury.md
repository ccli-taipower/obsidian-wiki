# 鋼琴技巧、表現力與傷害預防

3 篇論文：1 篇技巧與表現力概覽、1 篇延伸技巧教學指南、1 篇傷害預防的生物力學分析程序。

## 1. Jia (2023) — 鋼琴演奏技巧與音樂表現力

### 技巧的歷史演變

| 時期 | 技巧特徵 |
|------|--------|
| **巴洛克/早期古典**（1600-1820）| 大鍵琴/翼琴為主；力度範圍有限；重視觸法、裝飾音、分句；手指獨立性 |
| **浪漫時期**（~1815-1910）| 鋼琴能力大幅擴展；全範圍力度、踏板效果、炫技段落；Liszt 和 Chopin 開創 |
| **炫技傳統** | 雙八度（double-octave）、三對四（three-against-four）節奏；要求技巧與精準度 |
| **20 世紀創新** | Debussy 和 Bartók 的預置鋼琴（prepared piano）、集群和弦（clusters）、非傳統觸鍵 |
| **當代** | 融合爵士、極簡、電子音樂；非傳統手位、節奏複雜度、數位增強 |

### 音樂表現力的 7 個面向

1. **技術基礎**：音階、琶音、手指練習培養力量與靈活度
2. **力度控制**：操控觸鍵力量，從 ppp 到 fff 的完整光譜；gradual crescendo/diminuendo
3. **踏板藝術**：延音踏板延長共鳴；半踏板和快速踏板換踩控制清晰度
4. **Rubato 與速度彈性**：「偷來的時間」——伸縮特定樂句注入個性與情感
5. **觸法與分句**：staccato、legato、portamento 定義音樂句子的輪廓
6. **情感投入**：連接作曲家意圖與個人共鳴；超越音符的情感表達
7. **詮釋性創意**：融合忠於原譜與個人想像力；跨風格融合

### 與 V6 DP 的關聯

V6 目前處理的是面向 1 的一個子集（手指跨度的生物力學最佳化）。面向 2-7 均未建模，但都可能影響最佳指法選擇。

## 2. Proulx (2009) — 延伸鋼琴技巧教學指南

### 核心內容

⭐ 延伸技巧（extended techniques）：直接在鋼琴內部弦上演奏，20 世紀初在美國興起。

**先驅者**：
- **Henry Cowell**（1897-1965）：*The Tides of Manaunaun*（1917, clusters）、*The Banshee*（1925, glissando/pizzicato）、*Sinister Resonance*（1930, mute/harmonic）
- **George Crumb**（b.1929）：結合傳統與非傳統技巧，包括人聲與打擊效果

### 延伸技巧分類

| 類型 | 說明 |
|------|------|
| **弦上技巧（String techniques）** | Pizzicato（撥弦）、glissando（滑奏）、mute（消音）、harmonics（泛音） |
| **預置鋼琴** | 在弦上放置異物改變音色（Cage 等） |
| **弓弦技巧** | 用弓在鋼琴弦上拉奏 |
| **打擊技巧** | 敲擊琴身、琴蓋等部位 |
| **人聲技巧** | 演奏同時唱歌、說話等 |
| **傳統鍵盤技巧的擴展** | clusters、拳頭、前臂等 |

### 大鋼琴構造知識

Proulx 詳細說明演奏延伸技巧需要的大鋼琴知識：琴殼（casework）、弦背（strung back）、弦（strings）、擊弦機構（playing mechanism）。不同鋼琴的弦段（string sections）位置不同，需要適應。

### 注意事項

- 身體傷害風險（尤其是弦上技巧）
- 鋼琴損壞風險
- 清潔問題
- 選擇練習用鋼琴（primary vs secondary）
- 部分鋼琴老師和場地反對在鋼琴上使用延伸技巧

### 與 V6 DP 的關聯

延伸技巧完全超出 V6 DP 的範疇——V6 假設傳統鍵盤觸鍵。但 Telles (2020) 在 `src_fingering_algorithms_batch2` 中討論的 martellato 技法（拳頭、單指大力觸鍵）與此相關。

## 3. Wristen (2000) — 避免鋼琴相關傷害：生物力學分析的理論程序

### 核心貢獻

⭐ 提出一套**定性生物力學分析程序**，供鋼琴教師和醫療人員觀察和評估鋼琴家的動作模式，識別潛在傷害風險。

### Meinke 的鋼琴家 4 大人體工學法則

1. **使用動量（momentum）輔助工作**
2. **使用平滑曲線運動，而非直線急促運動**
3. **使用最適合的肌肉群完成工作**
4. **避免手腕偏離中性位置**

### 傷害風險動作

**應避免的動作**：
- 持續小幅重複動作
- 長時間尺偏（ulnar deviation）——尤其在鍵盤兩端
- 按鍵到底後仍施加壓力（bottoming out）
- 手/手指持續過度伸展
- 拇指上「搗」（pounding）
- 靜態手位
- 手腕沉到中性以下
- 突然停止或改變運動方向
- 手指獨立抬高而不借助較大肌肉群

**應促進的動作**：
- 平滑、協調的動作
- 隨動（follow-through）向上的動作
- 盡可能回到手腕中性位置
- 用大臂長弧線運動代替手腕每個音彈跳
- 呼吸與音樂分句協調

### 7 個運動模式清單（Charts 1-7）

Wristen 為 7 種基本鋼琴技巧提供了**詳細的生物力學清單**，描述從脊椎到指尖的每個身體部位在執行該技巧時的正確運動模式：

| 清單 | 技巧 | 關鍵動作描述 |
|------|------|----------|
| **Chart 1** | 音階 | 手腕微幅垂直和橫向調整；拇指穿越時前臂輕微旋前；手指自然彎曲，不施加多餘力量 |
| **Chart 2** | 琶音 | 類似音階但手指更分開、手弓更高；肘部離軀幹更遠 |
| **Chart 3** | 顫音（Trills）| 手腕微屈以利骨骼和關節穩定；小幅手指動作；過多前臂旋轉或手臂重量會使顫音減慢或停止 |
| **Chart 4** | 雙三度音階 | 手掌相對張開但不僵硬；手腕維持直線 |
| **Chart 5** | 八度音階 | 手持續處於尺偏狀態；拇指連續使用使手弓變平；前臂和手腕稍高於前臂 |
| **Chart 6** | 分解和弦 | 橫向調整動作；小手需要更多橫向手腕運動 |
| **Chart 7** | 分解八度 | 前臂交替旋前/旋後；手腕幾乎無垂直運動 |

### 手部大小的影響

- 手越小，橫向調整動作（尺偏和橈偏）越明顯
- 速度越慢，調整動作越明顯可見；快速時動作變小甚至不可見
- 清單應作為觀察指南而非強制標準——僵硬套用反而可能造成緊張

### 傷害預防建議

| 建議 | 說明 |
|------|------|
| 絕不忍痛練習 | — |
| 避免突然增加練習時間/強度 | — |
| 分段練習（≤30 分鐘） | 交替不同曲目 |
| 全天分散練習 | — |
| 暖身和收操 | — |
| 手冷時不要彈琴 | 寒冷增加傷害風險 |
| 注意經驗與傷害的反比關係 | 學新技巧時更脆弱 |
| 參加一般體能訓練 | — |

### 與 V6 DP 的關聯

| 發現 | V6 DP 啟示 |
|------|----------|
| 手腕中性 = 安全+高效 | V6 的 span_cost 懲罰過度伸展——符合 Meinke 法則 4 |
| 拇指穿越需前臂旋前 | V6 的 transition_cost 已考慮拇指穿越方向規則 |
| 八度音階手持續尺偏 | V6 不建模手腕姿態，但 Chart 5 的描述可作為未來和弦指法的參考 |
| 小手需要更多橫向調整 | 支持 concept_small_hands 的跨度縮放建議 |
| 清單不應僵硬套用 | 與 Talking Fingers「程序性知識主導」一致——規則永遠有例外 |
